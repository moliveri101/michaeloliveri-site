DOCUMENTS
=========

PDFs and other files you want to link from the site.

Folders
-------
cv/           CV PDFs. Suggested filename: oliveri-cv-YYYY-MM.pdf
press/        Press clippings, reviews, articles (PDF or image scans).
statements/   Artist statements, project descriptions, essays.
technical/    Tech specs, installation guides, rider docs for venues.

Wiring into HTML
----------------
In the Bio / CV section, add a link like:
  <p><a href="documents/cv/oliveri-cv-2026-04.pdf" target="_blank">
    Download CV (PDF)
  </a></p>

Anywhere else you want to surface a document, use the same pattern.
All files in these folders are public once the site is deployed — only
put things here you want the world to see.
